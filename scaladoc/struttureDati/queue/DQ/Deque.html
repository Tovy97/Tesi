Deque 

Companion object Deque

final case class Deque[+E](sx: List[E], dx: List[E]) extends Queue[E] with Product with Serializable

Implementa le code doppie, che sono code in cui le operazioni di inserimento, lettura e cancellazione si possono compiere su entrambi gli estremi.

E

indica il tipo di elementi contenuti nella coda doppia. È covariante.

sx

è la parte iniziale della coda doppia (testa).

dx

è a parte finale della coda doppia (coda).
